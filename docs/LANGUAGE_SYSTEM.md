🌍 نظام إدارة اللغات العالمي - Tribly Global Language System

═══════════════════════════════════════════════════════════════════════════════

📋 نظرة عامة:

تم تطوير نظام متعدد اللغات شامل يدعم 3 لغات:
✅ العربية (ar) - RTL
✅ الإنجليزية (en) - LTR
✅ الفرنسية (fr) - LTR

═══════════════════════════════════════════════════════════════════════════════

⚙️ كيفية العمل:

1. ملف js/language.js
   └─ يحتوي على فئة LanguageManager
   └─ تدير جميع الترجمات والتبديل بين اللغات
   └─ يحفظ اللغة المختارة في localStorage

2. مفتاح اللغات في Navbar
   └─ 3 أزرار (AR - EN - FR)
   └─ يتم إنشاؤها تلقائياً في أعلى الصفحة
   └─ تبديل فوري للغة دون إعادة تحميل

3. نظام البيانات (data-i18n)
   └─ أضف data-i18n="key" لأي عنصر
   └─ سيتم تحديث النص تلقائياً عند التبديل

═══════════════════════════════════════════════════════════════════════════════

🚀 الاستخدام:

للعناصر التي تحتوي على نص:

```html
<h1 data-i18n="hero.findYourTribe">Find Your</h1>
<button data-i18n="nav.join">Join</button>
```

للعناصر التي تحتوي على placeholder:

```html
<input type="text" data-i18n="profile.name" placeholder="Name">
```

لتحديث النص برمجياً:

```javascript
window.languageManager.setLanguage('en');
```

للحصول على ترجمة:

```javascript
const text = window.languageManager.get('nav.join');
```

🔧 إضافة لغة جديدة:

1. افتح js/language.js

2. أضف اللغة الجديدة للمتغير translations:

```javascript
sp: { // Spanish
  'nav.howItWorks': 'Cómo funciona',
  'nav.about': 'Acerca de',
  // ... باقي الترجمات
}
```

3. أضف الرمز لمفتاح اللغات:

ابحث عن:
```javascript
const languages = ['ar', 'en', 'fr'];
```

وغيّره إلى:
```javascript
const languages = ['ar', 'en', 'fr', 'sp'];
```

💾 التخزين:

اللغة المختارة تُحفظ في localStorage تحت مفتاح "language"

الحصول على اللغة الحالية:
```javascript
const currentLang = localStorage.getItem('language');
```

مسح اللغة المحفوظة:
```javascript
localStorage.removeItem('language');
```

```javascript

});
```
